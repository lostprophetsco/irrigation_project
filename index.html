Система автоматического полива

Проект irrigation system для газона и грядок

📋 Обзор проекта

Полноценная система автоматического полива для газона (74 кв.м) и грядок (27 кв.м) с использованием современного оборудования Hunter.

🌱 Основные характеристики

  • Площадь полива: 101 кв.м (74 кв.м газон + 27 кв.м грядки)
  • Количество зон: 4 (3 для газона + 1 для грядок)
  • Контроллер: Hunter ELC-401i-E (4 станции)
  • Насос: Вихрь ПН-650 (0.65 кВт, 55 л/мин)
  • Дождеватели: MP1000 (1 шт), MP2000 (2 шт)
  • Стрип-спринклеры: Hunter SS-530 (2 шт)
  • Бак: 500-750 литров

🔧 Оборудование

  • Контроллер: Hunter ELC-401i-E Eco-Logic
  • Датчик влажности: Hunter SOIL-CLIK
  • Поплавковый датчик: Защита от сухого хода
  • Фильтрация: Двухступенчатая (грубая + тонкая)
  • Реле давления: 2.0-3.5 бар
  • Распределительные коробки: 2 шт. (3-клапанная для газона, 1-клапанная для грядок)

💡 Особенности системы

  • Умный полив: Датчик влажности почвы блокирует полив при достаточной влажности
  • Защита насоса: Поплавковый датчик предотвращает сухой ход
  • Автоматизация: Реле давления обеспечивает автоматическую работу насоса
  • Надежность: Двухступенчатая фильтрация защищает оборудование
  • Организация: Две распределительные коробки: первая на 3 клапана для газона, вторая на 1 клапан для грядок
