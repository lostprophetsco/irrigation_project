Спецификация ПНД фитингов и переходников

📝 Общая информация

Вся система собирается на ПНД трубах и фитингах. Для соединения с металлическими компонентами (насос, фильтры, клапаны) используются переходники МПН (металл-полиэтилен).

⚠️ Важные примечания

1. Тип фитингов: Рекомендуются компрессионные фитинги для простоты монтажа и демонтажа. Сварные фитинги требуют специального оборудования и не рекомендуются для систем с частым обслуживанием.

2. Переходники МПН: Используйте качественные латунные переходники МПН для надежного соединения ПНД труб с металлическими компонентами.

3. Давление: Все фитинги должны быть рассчитаны на рабочее давление не менее 6 бар (с запасом относительно рабочих 2-3 бар).

4. Глубина прокладки: Подземные трубы прокладываются на глубине 30-50 см для защиты от механических повреждений и замерзания.

5. Консервация: Дренажный клапан устанавливается в нижней точке системы для полного слива воды на зиму.

✅ Итоговая спецификация

Трубы ПНД: ~56-75 м суммарно (40 мм, 25 мм, 20 мм + 16.1м дренаж)

Фитинги ПНД компрессионные: ~30-35 шт. (включая дренаж, 4 клапана)

Переходники МПН: ~12-14 шт. (включая предохранительный клапан)

Латунные фитинги и ниппели: ~17-20 шт. (включая предохранительный клапан)

Хомуты крепежные: ~19 шт.

Дренажные клапаны: 3 шт. (шаровые 20 мм)

Дренажные колодцы: 3 шт. (пластиковые минимальные 160 мм × 600 мм)

Предохранительный клапан: 1 шт. (латунный 1", 5 бар)

Уплотнительные материалы: ФУМ-лента, лен, паста, тефлоновая лента

💡 Рекомендации по закупке

1. Бренд: Рекомендуется приобретать фитинги одного производителя для обеспечения совместимости.

2. Запас: Рекомендуется приобретать фитинги с запасом 10-15% на случай ошибок монтажа.

3. Качество: Используйте только сертифицированные фитинги для обеспечения герметичности и долговечности системы.

4. Инструменты: Для компрессионных фитингов потребуется ключ для ПНД труб (обычно 32-40 мм).
